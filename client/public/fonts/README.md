# Fonts for PDF export

To display **Arabic and English** text correctly in the Products Report PDF (instead of unknown characters), place a Unicode font file here:

- **File name:** `NotoSansArabic-Regular.ttf`
- **Download:** [Google Fonts – Noto Sans Arabic](https://fonts.google.com/noto/specimen/Noto+Sans+Arabic) → download the ZIP and copy `NotoSansArabic-Regular.ttf` into this folder.

If this file is present, the app will use it for PDF export. Otherwise it will try to load the font from a CDN (requires internet and CORS).
